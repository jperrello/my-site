Saturn

Zero-configuration AI service discovery system that draws inspiration from Apple's Bonjour software. AI services broadcast availability on your local network, devices find them automatically, with no manual setup required. Allows organization of all cloud and local providers into one centralized server that any user on the network can connect to and use. Uses mDNS/DNS-SD (Bonjour inspiration) to broadcast available AI services with priority-based routing. Run Ollama locally as priority 10, OpenRouter in the cloud as priority 50, and clients automatically connect to the best available option with seamless failover when services go offline. Everything speaks OpenAI-compatible API regardless of backend, so clients work with any provider. FastAPI servers register themselves on the network, clients discover them via either native DNS-SD commands (works in any language, zero dependencies) or the Python zeroconf library. Built multiple server implementations (OpenRouter proxy with 200+ models, Ollama local inference, fallback server for testing), various clients (simple chat, file upload with multimodal support for images/PDFs, local proxy that bridges third-party apps), and integrations for Open WebUI and VLC. The core idea is trading manual service configuration for automatic network-based discovery. Ultimately spend lless time configuring, more time building.

Python FastAPI Zeroconf/mDNS Service Discovery

Language-Driven Play: LLMs as Game-Playing Agents

Research project investigating whether Large Language Models can effectively evaluate procedurally generated game content. Built on MiniStS (a simplified Slay the Spire implementation) and tested multiple agent architectures: Reverse Chain-of-Thought (RCoT), Chain-of-Thought (CoT), Monte Carlo Tree Search (MCTS), and zero-shot LLM agents. Extended the original paper by comparing modern LLM providers (GPT-4.1, Claude Sonnet 4.5, Gemini 2.5) via OpenRouter. Integrated grammar-based card generation system (GIGL) from collaborating team to test agents on novel, procedurally generated content the models had never seen. Results showed RCoT agents significantly outperformed heuristic scripts and search-based methods by semantically understanding card text and long-term strategy, achieving 50.6 avg health vs MCTS's 35.6 in strategic scenarios. Interesting finding: LLMs struggled with their own generated content—traditional Backtrack-Depth3 agents performed better on GIGL cards than RCoT. Comprehensive evaluation across 5 scenarios with 25 simulations each, demonstrating that structured reasoning is critical for general game playing. Academic paper published as part of graduate coursework.

Auto-Student: Academic Assignment Automation System

Academic research project for graduate coursework exploring automated assignment completion using AI to expose plagiarism vulnerabilities in educational systems. Integrates Canvas LMS and OpenAI APIs to automatically retrieve assignments, process multiple file formats (PDFs, HTML, documents), extract YouTube transcript data from embedded videos, and generate responses using GPT models. Implemented async/await patterns to handle large assignment batches without timeouts and built intelligent content summarization to manage context window constraints. Streamlit interface provides real-time progress tracking and submission previews. Demonstrates critical security gaps in current academic integrity systems and the need for improved plagiarism detection. Built as a proof-of-concept for educational research, not for actual use.
